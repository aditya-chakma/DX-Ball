# DX - Ball  <img src='img/dx.png' style='height:1.5em;'/>

### Features:
- 5 Levels
- Lives system
- Scoring system

### Requirements
- g++ compiler
- OpenGL/glut library

### How to run
- Put the glut libraries in lib/ directory
- Ubuntu:
  - Compile: `g++ final.cpp dx-ball`
  - Run: `./dx-ball`
- Windows:
  - Hit Compile on IDE (Codeblocks)
  - Run
